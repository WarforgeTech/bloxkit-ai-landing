# **App Name**: The Fortnite UEFN Bootcamp

## Core Features:

- Responsive Layout: Implement a scrollable, single-page design optimized for mobile responsiveness.
- Social Proof Section: Showcase video testimonials from successful students and integrate trust badges.
- Interactive Content: Use accordion-style FAQ and curriculum sections to manage content effectively.
- Call-to-Action Optimization: Include multiple, strategically-placed 'Enroll Now' CTAs to increase conversion rates.
- AI Island Concept Tool: Integrate AI-powered design suggestions tool to brainstorm creative island concepts. This is a tool.
- Scrolling Transitions: Create smooth transitions between content sections to keep the user engaged.

## Style Guidelines:

- Primary Color: Electric Blue (#5FCEEA) – Lighter Fortnite-inspired blue for headlines, links, and Fortnite-themed elements; evokes energy, creativity, and trust in tech/gaming education.
- Background Color: Dark Gray (#28282B) – Modern, eye-friendly dark base for sections; provides high contrast with light text and gold accents, common in gaming sites like Unreal Engine for immersion and reduced strain.
- Accent Color: Fortnite-Friendly Gold/Yellow Accent (#FFD447) (Solar Gold) – Bright, eye-catching highlight for revenue stats, CTA buttons, and "New" badges or key highlights; replaces Neon Green and symbolizes value and excitement.
- Secondary/Neutral Colors: White (#FFFFFF) for body text, subheads – Clean readability on dark bg. Light Gray (#D3D3D3) for subtle accents, borders, or muted backgrounds. Optional Fortnite Pop: Deep Purple (#9D4DBB) for icons/graphics (e.g., UEFN tools) to add gaming vibrancy without overwhelming tech focus.
- Usage Rules: Use primary blue for headers and links. Use #FFD447 for exciting elements like revenue stats, CTAs, and key highlights. Neutrals for body text and subtle elements. Ensure text on dark background passes contrast checks, especially with the gold accent. Avoid over-saturation; test on colorblind simulators.
- Headlines: 'Space Grotesk' (bold, futuristic sans-serif) – Sizes: H1 48-64px, H2 32-48px; uppercase for impact, reflecting Fortnite's bold logos and tech dev tools.
- Body Text: 'Inter' (clean sans-serif) – Sizes: 16-18px base, 14px for small notes; high legibility for tutorials/code snippets.
- Pairing Tips: Limit to 2-3 weights (bold for emphasis, regular for paragraphs). Line height 1.5-1.6 for readability. Use blue accents for links/CTAs.
- Custom icons: Fortnite-style (e.g., blocky assets, UEFN devices like checkpoints/NPCs) in primary blue or the gold accent; use simple vectors for fast loading.
- Graphics: Screenshots/GIFs from course (e.g., island builds, AI tools) with subtle glows/neon borders. Hero video bg: Fortnite-inspired vibrant island overlay on dark gray.
- Avoid clutter: Space elements with 20-40px padding; use for benefits bullets or curriculum icons.
- Single-page scroll: Clear sections with subtle dividers (light gray lines or blue gradients). Hero above fold: Video + headline/CTA dominant.
- Responsiveness: Fluid grid (e.g., 1200px max width); stack sections on mobile, enlarge CTAs to 48px+.
- Whitespace: Generous (24-48px between elements) for tech users scanning content.
- Hover: Scale CTAs 1.1x with a subtle gold glow; fade-in icons on scroll for engagement without distraction.
- Transitions: Smooth 0.3s ease for section reveals; avoid heavy effects (e.g., no particle explosions) to suit programmers/basic tech skills.
- Performance: Optimize for <2s load; lazy-load images/videos.
